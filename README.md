Portfólio Pessoal - Mauricio Petri Junior

Este projeto consiste em uma página web desenvolvida com o objetivo de apresentar um portfólio pessoal na área de tecnologia da informação. A página reúne informações básicas sobre o estudante, incluindo uma breve descrição pessoal, habilidades técnicas e comportamentais e uma lista de projetos desenvolvidos ou idealizados ao longo da formação.

A proposta do projeto é demonstrar conhecimentos fundamentais de desenvolvimento web utilizando tecnologias básicas do front-end. A página foi desenvolvida utilizando HTML para a estruturação do conteúdo, CSS para estilização visual e JavaScript para adicionar pequenas interações e comportamento dinâmico.

O portfólio funciona como uma landing page simples que pode ser utilizada como base para apresentação profissional inicial na área de desenvolvimento de software. A estrutura do projeto foi organizada de forma clara, separando os arquivos de marcação, estilo e script, seguindo boas práticas de desenvolvimento web.

Além disso, o projeto também serve como exercício prático de organização de código, estrutura de arquivos e documentação de projetos, habilidades importantes no contexto profissional da área de tecnologia.

Tecnologias utilizadas no projeto:

HTML
CSS
JavaScript

O projeto pode ser facilmente expandido no futuro com novos projetos, melhorias visuais, integração com APIs e publicação online em plataformas como GitHub Pages.
